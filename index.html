RESTful/JSON API

支持浏览器和桌面客户端

交易系统使用一个通用的交易后端API，使用了RESTful的风格和JSON编码，适应现代应用开发的模型。网页端所有操作均使用AJAX技术，保证了操作的流畅性和网站整体性能。通过将API和渲染层分离，使得系统的架构设计足够简洁，同时也使得该平台具有较高的可扩展性。

高性能后端

全异步编程范式，高性能和高扩展性

使用完全异步的编程范式，支持上万同时在线的用户数量。通过将应用服务器进行负载均衡，可以很容易地将该平台扩展到百万级别的同时在线用户数量，能够满足在现实情况下对系统性能的极限要求。
